Creating Material UI Forms

back to topics

Creating Material UI forms is done just like traditional Sencha controls except the controls are Material UI.
Ideal Forms implements a core layout below both Sencha and Material UI, as such, each control is encapsulated and managed automonously. So, your forms can contain a combination of Sencha ExtJs controls and Material UI controls. The following list shows the available material UI contols:

AppBar
Button
CardList
CommentList
CommentSuperList
Datebox
Drawer
FormButtons
Grid
Html
HistoryList
Icon
Radio
Select
Textarea
