Work Log Reporting - Installation

Back to Summary   Back to Library

Requirements:
Scriptrunner - common JIRA plugin, provides a button link on Jira search screen.

Installation Steps:
Step 1: Install form group:
  1. Download form group - worklog form group
  2. Click "Forms Admin", and install the form group using "Upload Group".
Step 2: Add button to Issue search screen using Scriptrunner:
  1.     Add web "Fragments" from Scriptrunner menu
  2.     Click "Create Script Fragments"
  3.     Click "Custom web item"
  4.     Name= "Worklog Report"
  5.     Section for button= "jira.navigator.pluggable.items"
  6.     Key= "ifj-worklog-report"
  7.     Menu text= "Worklog"
  8.     Set link to be the following *
* The link item needs to navigat to the form and pass to it the current JQL or filter if set. This is done with this line of code: 
javascript:if(jql.value){window.location="/plugins/servlet/iforms?decorator=general&formId=Worklog%20Report&jql=" + jql.value} else {window.location="/plugins/servlet/iforms?decorator=general&formId=Worklog%20Report&jql=" + window.location.search.replace("?","")}
