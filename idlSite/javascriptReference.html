Javascript Reference

Ideal Forms use Javascript to enhance the functionality of the JIRA system. Atlassian JIRA has a fantastic set of Restful API calls that are fully exposed and available for your use. You can use Javascript to not only implement complex business rules but you can also use the JIRA API to perform any function you can dream up.

Javascript business rules (and style settings) are loaded into your form page for access by your form's events. The javascript functions are stored in the "Form Group" so all forms owned by a Form Group can leverage the functions and you can create common functions that all Form Group forms can reference.

All javascript rules are loaded into an Ideal Forms namespace. If you are looking for your functions in the console please look at:

Style Reference

back to topics

Ideal Forms provides a mechanism for setting and overriding styles using standard CSS. The way to do this is to add one or more Javascript business rules named "style". Doing this will load the styles into the web page. These styles are stored in the context of the Form Group and will load for all forms owned by the Form Group.
