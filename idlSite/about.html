About Ideal Forms for JIRA

Ideal Forms for JIRA is a plugin for Atlassian JIRA software. Server version only at the moment. The plugin is available here: https://marketplace.atlassian.com/plugins/com.idealfed.poc.idealforms/server/overview

Licensing: Ideal Forms for JIRA is released under the Atlassian software license.

3rd Party Licensing: Ideal Forms for JIRA uses several javascript libraries. These are:

  • jQuery: https://jquery.com/
  • Moment: https://momentjs.com/
  • FileSaver: https://github.com/eligrey/FileSaver.js/
  • Sencha ExtJs: https://www.sencha.com/legal/gpl//
  • Idealfed Forms Utilities for ExtJs: https://github.com/pmalone13/idealFedFormsEngine/
